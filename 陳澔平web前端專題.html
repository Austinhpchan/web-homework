幼兒教育 學類

學類介紹

這個學類學什麼？

幼兒教育的核心為關注2-6歲學齡前幼兒的發展和學習，以統整性課程培養幼兒具有覺知辨識、表達溝通、關懷合作、推理賞析、想像創造以及自主管理的能力素養。

專業學科的學習內容大致涵蓋幼兒教保、幼兒園課程設計與教材教法、幼兒發展與學習、幼兒學習環境設計、特殊幼兒教育、音樂與藝術、文學與戲劇。

如果你學習幼兒教育學類，你可以享受和孩子一起生活、一起創作和解決問題的樂趣，並且看見孩子的成長。

相似學類比一比

本學類主要培育「幼兒教育及照顧法」中指稱之「教保服務人員」的「幼兒園教師」與「教保員」。

本學類容易與教育學類搞混。

教育學類：主要培養國小及以上的師資、教育行政與學術研究人才。

就要你這款

就要你喜歡帶著好奇探索世界、喜歡遊戲中學習並且從事和人有關的工作，而且喜歡又能欣賞年幼的孩子，常常願意和他人合作溝通，以正向的態度面對問題、解決問題，那麼幼兒教育學類會是合於你特質的選擇。

必修或核心課程？

●教學基本學科：幼兒文學、幼兒藝術、幼兒體能、幼兒戲劇

●教育基礎課程：教育概論、教育心理學、教育哲學、教育社會學

●教育方法課程：教學原理、幼兒園課程設計、幼兒遊戲、幼兒學習環境設計、幼兒園行政

●教學實習課程：幼兒園教學實習

●教保專業知能課程：幼兒發展、幼兒觀察、幼兒園教保活動課程設計、幼兒園教材教法、特殊幼兒教育

生涯進路

幼教老師

具備教師資格，在幼教場所擔任教師，設計與執行促進兒童語言、身心、體格發展的各種教學活動。

(104薪資情報: 3.1萬～4.0萬/幼教班老師 )

生涯進路

教保員

在幼教場所協助執行促進兒童語言、身心、體格發展的各種教學活動。

(104薪資情報: 3.1萬～3.8萬/教保員 )

生涯進路

保育員

於安親之教學場所，從事幼兒照顧及教養工作。

(104薪資情報: 3.1萬～3.8萬/教保員 )

生涯進路

幼稚園園長

負責園區硬體的規劃與改善、人事管理，並照護幼兒學習安全及處理家長對園所的相關問題。

(104薪資情報: 3.1萬～4.0萬/幼教班老師 )

生涯進路

托嬰保母人員

受雇負責嬰幼兒看護、教導幼童學習及照顧其日常生活。

(104薪資情報: 3.1萬～3.7萬/托育員 )
